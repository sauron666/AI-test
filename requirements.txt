# ═══════════════════════════════════════════════════════════════════
#  SAURON — runtime requirements
#
#  Philosophy:
#    - Use LOWER bounds (>=) so pip picks the newest wheel for the
#      active Python (3.11 → 3.13+). Hard == pins caused source
#      builds that broke on Python 3.13.
#    - Upper bounds only where a known breaking change exists.
#    - Optional extras (weasyprint, asciinema) are in
#      requirements-optional.txt so the core install never fails
#      because of a missing system library like Cairo.
# ═══════════════════════════════════════════════════════════════════
